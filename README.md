# Tool-js

#### 这不是Node.js包！

**这是个javascript的小工具？**

# 用法

## tool.js

```HTML
<script src="https://cdn.jsdelivr.net/gh/Fgaoxing/Tool-js/tool.js"></script>
```

会有一定延迟

### getQueryVariable

```JavaScript
getQueryVariable(variable,err)
```
用于获取url参数

| variable | err |
| --- | --- |
| 参数名 | 不存在时返回 |

### lantern

```JavaScript
lantern()
```
春节小灯笼

### Memorial

```JavaScript
Memorial()
```
国家公祭日时，网站变灰

### TombSweepingDay

```JavaScript
TombSweepingDay()
```
清明节时，网站变灰
